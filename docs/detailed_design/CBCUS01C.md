# CBCUS01C 詳細設計書

## 1. 概要

`CBCUS01C` は、顧客マスタ (`CUSTFILE`) から全レコードを順次読み込み、それぞれの内容を標準出力（`DISPLAY`）に表示するユーティリティプログラムです。マスタメンテナンス後の確認や、簡単なデータダンプとして使用されます。

## 2. プログラム情報

- **プログラム ID**: `CBCUS01C`
- **作成者**: AWS
- **タイプ**: BATCH COBOL Program
- **関連 JCL**: `CUSTLIST.jcl` (想定)

## 3. 入出力定義

### 3.1. 入力ファイル

- **CUSTFILE-FILE**: 顧客マスタ (VSAM KSDS)。全件読込。

### 3.2. 出力ファイル

- なし（標準出力への表示のみ）

## 4. 処理ロジック詳細

### 4.1. 主処理ループ

`CUSTFILE` を物理順（顧客 ID 順）に最後まで読み込みます。

### 4.2. 表示処理

読み込んだ 500 バイトの顧客レコード (`CUSTOMER-RECORD`) を、そのまま `DISPLAY` 文により出力します。

## 5. エラーハンドリング

- ファイルオープン、読込、またはクローズエラー時に、エラー詳細を表示してプログラムをアベンド (`CEE3ABD`) させます。
- エラーハンドリング用サブルーチンとして `Z-DISPLAY-IO-STATUS` および `Z-ABEND-PROGRAM` が定義されています。

## 6. 特記事項

- 本プログラムはデータ抽出ではなく、内容の可視化を目的としたシンプルな設計となっています。
