# CBACT02C 詳細設計書

## 1. 概要

`CBACT02C` は、カードマスタ (`CARDFILE`) から全レコードを順次読み込み、それぞれの内容を標準出力（`DISPLAY`）に表示するユーティリティプログラムです。主にデータの確認やデバッグ目的で使用されます。

## 2. プログラム情報

- **プログラム ID**: `CBACT02C`
- **作成者**: AWS
- **タイプ**: BATCH COBOL Program
- **関連 JCL**: `CARDLIST.jcl` (想定)

## 3. 入出力定義

### 3.1. 入力ファイル

- **CARDFILE-FILE**: カードマスタ (VSAM KSDS)。全件読込。

### 3.2. 出力ファイル

- なし（標準出力への表示のみ）

## 4. 処理ロジック詳細

### 4.1. 主処理ループ

`CARDFILE` を物理順（カード番号順）に最後まで読み込みます。

### 4.2. 表示処理

読み込んだ 150 バイトのカードレコード (`CARD-RECORD`) を、そのままヘッダー等なしで `DISPLAY` 文により出力します。

## 5. エラーハンドリング

- ファイルオープンまたは読込エラー時に、エラー詳細を表示してプログラムをアベンド (`CEE3ABD`) させます。

## 6. 特記事項

- `CBACT01C` のような外部ファイルへの書き出しロジックは含まれておらず、純粋なリスト表示プログラムとなっています。
