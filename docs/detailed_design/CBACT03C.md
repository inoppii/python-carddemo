# CBACT03C 詳細設計書

## 1. 概要

`CBACT03C` は、カード・アカウントクロスリファレンスマスタ (`XREFFILE`) から全レコードを順次読み込み、それぞれの内容を標準出力（`DISPLAY`）に表示するユーティリティプログラムです。`CBACT02C` と同様、データの確認やデバッグを目的としています。

## 2. プログラム情報

- **プログラム ID**: `CBACT03C`
- **作成者**: AWS
- **タイプ**: BATCH COBOL Program
- **関連 JCL**: `XREFLIST.jcl` (想定)

## 3. 入出力定義

### 3.1. 入力ファイル

- **XREFFILE-FILE**: クロスリファレンスマスタ (VSAM KSDS)。全件読込。

### 3.2. 出力ファイル

- なし（標準出力への表示のみ）

## 4. 処理ロジック詳細

### 4.1. 主処理ループ

`XREFFILE` を物理順（カード番号順）に最後まで読み込みます。

### 4.2. 表示処理

読み込んだ 50 バイトのクロスリファレンスレコード (`CARD-XREF-RECORD`) を、そのまま `DISPLAY` 文により出力します。

## 5. エラーハンドリング

- ファイルオープンまたは読込エラー時に、エラー詳細を表示してプログラムをアベンド (`CEE3ABD`) させます。

## 6. 特記事項

- 本プログラムはシンプルなリスト表示機能のみを提供しており、外部ファイルへのデータ抽出ロジックは含まれていません。
