# p_CBCUS01C 詳細設計書

## 1. 概要

`p_CBCUS01C.py` は、PostgreSQL の `customers` テーブルから全レコードを読み込み、標準出力（ログ）に表示するユーティリティプログラムです。

## 2. プログラム情報

- **プログラム ID**: `p_CBCUS01C.py`
- **言語**: Python 3.10+
- **接続DB**: Cloud SQL for PostgreSQL

## 3. 入出力定義

### 3.1. 入力 (Database)

- **テーブル**: `customers`
- **全件取得**: `SELECT * FROM customers ORDER BY cust_id`

### 3.2. 出力

- **標準出力**: 顧客レコードの内容をテキスト（または JSON 形式）でログ出力。

## 4. 処理ロジック詳細

1. データベース接続の確立。
2. `customers` テーブルから全レコードを取得。
3. 取得した各レコードについて、ログ出力ライブラリを使用して内容を表示。

## 5. エラーハンドリング

- DB 接続失敗時は例外をキャッチし、エラーログを出力して異常終了。
